# HashCalc: MD5, SHA, HMAC 📱🔑

[![Google Play Downloads](https://img.shields.io/badge/Google%20Play-50K%2B%20Downloads-green?logo=google-play&logoColor=white)](https://play.google.com/store/apps/details?id=com.goyalsoftech.hashcalc)

**HashCalc** is a fast, secure, and completely offline cryptographic utility for Android. It allows you to compute and verify message digests (hashes) and keyed-hash message authentication codes (HMAC) instantly on your mobile device. 

Trusted by **50,000+ users on Google Play**, HashCalc handles everything from quick text strings to massive multi-gigabyte files with enterprise-grade stability.

---

## 🚀 Core Features

* **🔑 Full HMAC Support:** Securely generate Hash-based Message Authentication Codes with support for both plain text and hexadecimal secret keys.
* **⚡ Comprehensive Algorithms:** Compute standard hashes and HMACs for **MD5**, **SHA-1**, **SHA-256**, **SHA-512**, **SHA3-256**, and **SHA3-512**.
* **🔍 Intelligent Hash Detector:** Easily identify unknown hash types based on their length, formatting, and character structure.
* **📊 Large File Support (4GB+):** Process heavy files and disk images smoothly with an active progress bar and "Stay Awake" logic to prevent system timeouts.
* **⏱️ History Log & Export:** Automatically tracks your last 50 calculations locally with built-in tools to export your verification logs.
* **🔄 Real-Time Comparison:** Paste an expected hash to cross-reference your results instantly with clear visual indicators (✔/✖).
* **🔒 100% Privacy & Ad-Free:** Runs completely offline. Zero advertisements, zero third-party tracking, and zero unnecessary device permissions.
